SysMonHUD

Privacy Protocol & Data Integrity System

Executive Summary

At ASTechAI, your privacy is hardcoded into our philosophy. SysMonHUD is a professional-grade telemetry tool designed to monitor system performance in real-time. We operate on a Local-First principle: your system data stays on your machine.

System Telemetry Audit

The application accesses hardware sensors to provide the following metrics. All data in this category is processed in volatile memory and never leaves your device.

Component Metrics Monitored Processing Type
CPU Load, Temperature, Clock Speed, Fan RPM, Voltage, Power (W) Local Only
GPU Core/Hotspot Temp, VRAM Usage, PCIe Throughput, Fan Speed Local Only
Network Real-time Download & Upload Speeds Local Only
Battery Voltage, Charge/Discharge Rate, Remaining Capacity, Health Local Only
Audio Peak Audio Levels (Visualizer mode only) Local Only

*Audio permission is used only to calculate visual peaks; no voice or sound is recorded or transmitted.

Required Capabilities

SysMonHUD requires specific system access to function correctly. Here is the transparency breakdown:

Platform Capability Justification
Windows WMI / PDH Access Fetching hardware sensor data (Temp, Load, RPM).
Android SYSTEM_ALERT_WINDOW Displaying the HUD overlay over other applications.
Android POST_NOTIFICATIONS Continuous speed monitoring in the notification shade.
Universal INTERNET License verification and Public IP check.

Data Integrity

We perform automated Integrity Checks to ensure the SysMonHUD binary has not been tampered with. This protects you from malware injections and ensures the metrics displayed are accurate and untainted.
